# Codex Build Instructions

This repository is a blueprint for a semi-complex educational quantum programming platform.

Before implementing code, read:

1. `README.md`
2. `docs/product-spec.md`
3. `docs/architecture.md`
4. `specs/file-structure.md`
5. `.codex/loops/README.md`

When using Next.js, read the relevant local Next.js documentation in `node_modules/next/dist/docs/` after dependencies are installed, because the parent workspace warns that this Next.js version may differ from older conventions.

Work loop by loop. Do not skip verification steps. Keep simulator math well tested before building dependent UI.
